# Element-plus-Vue3-Vite

#### 介绍

使用 Element-plus + Vue3 + Vite 搭建的后台 Pc 模板，
对 Table 进行封装，集成 hooks 表单
使用Mock模拟请求数据

#### 软件架构

1. vite
2. vue3
3. vuex
4. vue-router
5. Element-plus
6. mock

#### 安装教程

node version >= 12.0.0

```
 yarn add
```

#### 使用说明

1.  开发环境运行

```
 yarn run dev
```

2.  打包

```
yarn run build
```
