# Element-plus-Vue3-Vite

#### Description

The background PC template built by element plus + vue3 + vite,

Encapsulate the table and integrate the hooks form

Mock request data using mock

#### Software Architecture

1. vite
2. vue3
3. vuex
4. vue-router
5. axios
6. Element-plus
7. mock

#### Installation

node version >= 12.0.0

```
 yarn add
```

#### Instructions


1.  开发环境运行

```
 yarn run dev
```

2.  打包

```
yarn run build
```
